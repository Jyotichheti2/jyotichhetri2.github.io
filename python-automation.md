# Project: Automating Cybersecurity Tasks with Python

## Project Overview
This project showcases my ability to use Python to automate repetitive security tasks, improving efficiency and reducing human error in security operations.

## Skills Demonstrated
* **Parsing Files:** Automating the reading and updating of allow/deny lists (e.g., updating an `allow_list.txt` based on a `remove_list`).
* **Algorithm Development:** Implementing logic to search through and filter large datasets for specific security indicators.
* **String & List Manipulation:** Using Python methods to clean and format security logs for better analysis.
